# 鳶尾花 (Iris) 資料集分析與 KNN 演算法

## 簡介
本專案針對鳶尾花 (Iris) 資料集，實現了基本的數據分析、視覺化以及 K 鄰近分類演算法 (K-Nearest Neighbors, KNN)。  
目的是讓使用者學習如何處理資料、視覺化分析，以及應用機器學習模型進行分類。

## 功能
- 載入 Iris 資料集
- 進行資料的前處理和標準化
- 使用視覺化工具分析資料特徵分佈
- 應用 KNN 演算法進行分類
- 輸出分類結果與模型評估報告
